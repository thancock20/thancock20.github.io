Front-end projects

These projects are written in HTML, CSS, and Javascript. All are hosted on Codepen.

Random Programming Quotes Machine

A random quotes generator, where all of the quotes are related to programming.

Pomodoro Timer

A timer based on the Pomodoro technique. The user can set the next work or break time while the timer is still running.

Simple Calculator



A simple javascript calculator with parentheses and a running total on a second line.

Local Weather

Uses geolocation to get the users current location, and displays the current weather from the OpenWeatherMap API.

Twitch Status

An app that shows whether or not certain users are streaming on Twitch.tv, and what they are streaming if they are.

Wikipedia Viewer


"Instant" Wikipedia search engine that displays findings as the user types.

Camper News


Displays the Camper News from freeCodeCamp in a nice Pinterest type layout.

Tic Tac Toe


A Tic Tac Toe game. The computer is smart enough to never lose. Only win or draw.

Simon Game

A Simon game. Remember the button order and press them. If the user beats 20 stages she wins. But watch out it gets faster.
